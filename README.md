# perets_website

Статический сайт ресторана PERETS для GitHub Pages.

## Что есть сейчас

- Главный экран с логотипом и фото ресторана.
- Каталог с категориями и карточками блюд.
- Блоки о ресторане, галерея и контакты.
- Бесплатные шрифты Manrope и Oswald из Google Fonts.

## Как открыть

Откройте `index.html` в браузере.

Для GitHub Pages выберите источник деплоя `Deploy from a branch`, ветку `main`
и папку `/root`. Главная страница сайта лежит в корне репозитория.

На этом этапе база данных не подключена. Данные меню находятся в `script.js`
и сделаны близко к структуре таблицы `menu_items`.
