# perets_website

Первый GUI-прототип сайта ресторана PERETS с онлайн-заказом.

## Что есть сейчас

- Главный экран с логотипом и фото ресторана.
- Каталог с категориями и моковыми карточками блюд.
- Отдельная страница корзины с количеством, итогом и демо-формой оформления.
- Страница входа и регистрации пользователя.
- Блоки о ресторане, галерея и контакты.
- Бесплатные шрифты Manrope и Oswald из Google Fonts.
- Страница сравнения бесплатных шрифтов `frontend/font-preview.html`.

## Как открыть

Откройте `frontend/index.html` в браузере.

На этом этапе база данных не подключена. Данные меню находятся в
`frontend/script.js` и сделаны близко к структуре таблицы `menu_items`.
Корзина хранится в `localStorage`; контакты для заказа предполагаются из профиля.
